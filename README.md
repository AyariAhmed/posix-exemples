## POSIX simple exemples using C pthread library

> To run pont1:
```bash
gcc -std=c90 -o out/pont1 pont1.c -lpthread && ./out/pont1
```

> To run pont2:
```bash
gcc -std=c90 -o out/pont2 pont2.c -lpthread && ./out/pont2
```

> To run metro:
```bash
gcc -std=c90 -o out/metro metro.c -lpthread && ./out/metro
```
